# 🏆 ScoreMash

**ScoreMash** is a full-stack web application that displays live, upcoming, and completed sports matches with details for teams, players, scores, and more.

---

## 🚀 Features

- View lists of **live, upcoming, and finished matches**  
- Match-level pages with **detailed statistics**  
- Dynamic views served using **EJS templates**  
- Backend routing for **matches, teams, players, and more**  
- Session support for **user interactions**

---

## 🛠️ Tech Stack

- **Node.js** – Server runtime  
- **Express** – Server framework  
- **MongoDB & Mongoose** – Database & ODM  
- **EJS** – Template rendering  
- **HTML, CSS, JavaScript** – Frontend UI  
- **Express Session** – Session management  

---
